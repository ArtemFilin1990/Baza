# 7.6. Втулки TAPER BUSH (конусные втулки для крепления на валу)

*Дата последнего обновления: 2026-01-10*

---

**Определение:**
Коническая втулка для крепления деталей (шкивов, звездочек, муфт) на валу без обработки шпоночного паза.

**Конструкция:**
- Коническое наружное отверстие
- Цилиндрическое внутреннее отверстие
- Разрез для зажима
- Резьбовые отверстия для монтажа/демонтажа

**Стандарты:**
- **Британский:** 1008, 1108, 1210, 1215, 2012, 2517, 3020, 3030, 3535, 4040, 4545, 5050
- **Метрический:** аналогичные размеры

**Обозначение:**
- Пример: **1210** — конус 1:12, диаметр отверстия 10-40 мм

**Преимущества:**
- Простота монтажа и демонтажа
- Не требуется шпонка
- Точное центрирование
- Возможность установки на гладкий вал
- Передача высоких крутящих моментов

**Монтаж:**
- Установка втулки в ступицу детали
- Совмещение отверстий для болтов
- Затяжка болтов для создания натяга
- Контроль момента затяжки

**Демонтаж:**
- Ослабление болтов
- Перестановка болтов в выталкивающие отверстия
- Затяжка для выталкивания втулки

**Применение:**
- Шкивы ременных передач
- Звездочки цепных передач
- Зубчатые колеса
- Муфты
