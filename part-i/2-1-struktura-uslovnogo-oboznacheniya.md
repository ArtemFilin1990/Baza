# 2.1. Структура условного обозначения

*Дата последнего обновления: 2026-01-10*

---

Условное обозначение подшипника по ГОСТ состоит из основного обозначения и дополнительных знаков (префиксов и суффиксов).

**Структура основного обозначения (до 7 цифр):**

```
X XX X XX
│ │  │ └─── Внутренний диаметр (2 цифры)
│ │  └───── Тип подшипника (1 цифра)
│ └──────── Серия диаметров (1 цифра)
└────────── Серия ширин (1 цифра)
```

**Полная структура с дополнительными знаками:**

```
ПРЕФИКС - ОСНОВНОЕ ОБОЗНАЧЕНИЕ - СУФФИКС
```

**Примеры:**
- `6205` — шариковый радиальный однорядный подшипник
- `2207` — шариковый радиальный двухрядный сферический
- `32310` — роликовый конический однорядный
- `N 308` — роликовый цилиндрический однорядный без внутреннего кольца
