*Дата последнего обновления: 2026-01-10*

## 3.1. Нагрузка на подшипники

**Типы нагрузок:**

1. **Радиальная нагрузка (Fr)** — действует перпендикулярно оси вращения
2. **Осевая нагрузка (Fa)** — действует вдоль оси вращения
3. **Комбинированная нагрузка** — сочетание радиальной и осевой

**Грузоподъемность подшипников:**

### Динамическая грузоподъемность (C)
- Нагрузка, которую подшипник может выдержать в течение 1 миллиона оборотов
- Измеряется в Ньютонах (Н) или килоНьютонах (кН)
- Используется для расчета ресурса подшипника

### Статическая грузоподъемность (C₀)
- Нагрузка, которую неподвижный подшипник может выдержать без остаточной деформации
- Важна для подшипников, работающих с медленным вращением или качательным движением

**Расчет ресурса подшипника:**

```
L₁₀ = (C / P)^p × 10⁶ оборотов
```

где:
- L₁₀ — номинальный ресурс (в оборотах)
- C — динамическая грузоподъемность
- P — эквивалентная динамическая нагрузка
- p — показатель степени (3 для шариковых, 10/3 для роликовых)

**Эквивалентная нагрузка:**

```
P = X × Fr + Y × Fa
```

где:
- X — коэффициент радиальной нагрузки
- Y — коэффициент осевой нагрузки
- Fr — радиальная нагрузка
- Fa — осевая нагрузка

## 3.2. Угол контакта и преднатяг

**Угол контакта (α)** — угол между линией действия нагрузки на тело качения и плоскостью, перпендикулярной оси подшипника.

**Типовые углы контакта для радиально-упорных подшипников:**
- **15°** — преимущественно радиальная нагрузка
- **25°** — универсальное применение
- **40°** — преимущественно осевая нагрузка

**Преднатяг** — предварительное осевое сжатие подшипников в узле для:
- Увеличения жесткости узла
- Устранения зазоров
- Повышения точности вращения
- Снижения вибраций

**Способы создания преднатяга:**
1. Осевое сжатие пружиной
2. Регулировочная гайка
3. Установка проставочных колец определенной толщины

**Применение:**
- Шпиндели станков
- Высокоточные механизмы
- Редукторы с высокими требованиями к жесткости

## 3.3. Предельная частота вращения

**Предельная частота вращения** — максимально допустимая скорость вращения подшипника без перегрева.

**Факторы, влияющие на предельную частоту:**
- Тип подшипника (шариковые быстроходнее роликовых)
- Размер подшипника (меньше размер — выше частота)
- Тип смазки (масло позволяет более высокие скорости, чем пластичная смазка)
- Нагрузка (большая нагрузка снижает допустимую скорость)
- Тип сепаратора (металлический лучше для высоких скоростей)

**Ограничения по скорости:**

| Тип подшипника | Относительная быстроходность |
|----------------|------------------------------|
| Шариковый радиальный | Очень высокая |
| Шариковый радиально-упорный | Высокая |
| Роликовый цилиндрический | Средняя |
| Роликовый конический | Средняя |
| Роликовый сферический | Низкая |
| Игольчатый | Средняя |

**Тепловой расчет:**
- При превышении предельной частоты возникает перегрев
- Необходим контроль температуры подшипникового узла
- Возможно применение специальных высокоскоростных подшипников

## 3.4. Комплекты подшипников (дуплекс, триплекс)

**Комплекты подшипников** используются для восприятия больших осевых нагрузок и обеспечения повышенной жесткости узла.

**Основные схемы установки:**

### Дуплекс (два подшипника)
- **О-образная** (спина к спине) — высокая жесткость, лучше для опрокидывающих моментов
- **Х-образная** (лицом к лицу) — меньшая жесткость
- **Тандем** (последовательно) — для больших осевых нагрузок одного направления

### Триплекс (три подшипника)
- Три подшипника, установленных последовательно
- Для очень высоких осевых нагрузок

**Обозначение схемы установки:**
- **DB** — спина к спине (Back-to-Back)
- **DF** — лицом к лицу (Face-to-Face)
- **DT** — тандем (Tandem)

**Применение:**
- Шпиндели металлообрабатывающих станков
- Колесные ступицы автомобилей
- Редукторы с высокими требованиями к жесткости

## 3.5. Смазка подшипников

**Функции смазки:**
- Снижение трения и износа
- Отвод тепла
- Защита от коррозии
- Снижение шума
- Удаление продуктов износа

**Типы смазочных материалов:**

### Пластичные смазки
- **Литиевые** — универсальные, температура -30...+120°C
- **Кальциевые** — водостойкие, температура -10...+60°C
- **Натриевые** — высокотемпературные, температура -10...+120°C
- **Комплексные** — широкий диапазон температур -40...+150°C

### Масла
- **Минеральные** — стандартное применение
- **Синтетические** — экстремальные температуры и нагрузки
- **Вязкость** — подбирается по скорости вращения и температуре

**Интервалы смазки:**
- Зависят от типа подшипника, скорости, нагрузки, температуры
- Для подшипников с уплотнениями (2RS, ZZ) — смазка заложена на весь срок службы
- Для открытых подшипников — регулярная подача или замена смазки

**Количество смазки:**
- Переизбыток смазки может вызвать перегрев
- Недостаток — ускоренный износ
- Оптимально — заполнение 30-50% свободного объема

## 3.6. Монтаж, посадки и демонтаж подшипников

### Посадки подшипников

**Посадка на вал:**
- **С натягом** (для вращающегося внутреннего кольца)
- **Переходная** (для качающегося движения)
- **С зазором** (для неподвижного внутреннего кольца)

**Посадка в корпус:**
- **С зазором** (для вращающегося наружного кольца)
- **Переходная** (для стандартного применения)
- **С натягом** (для неподвижного наружного кольца при вибрациях)

### Правила монтажа

1. **Подготовка:**
   - Проверка чистоты вала и корпуса
   - Контроль размеров посадочных мест
   - Очистка от консервационной смазки

2. **Способы монтажа:**
   - **Холодный** — запрессовка с использованием оправки
   - **Горячий** — нагрев подшипника до 80-100°C в масляной ванне или индукционным нагревателем
   - **Механический** — с помощью гидравлических съемников

3. **Важные правила:**
   - Не допускать ударов по телам качения
   - Усилие прикладывать к тому кольцу, которое имеет натяг
   - Контролировать соосность при монтаже

### Демонтаж подшипников

**Инструменты:**
- Механические съемники (2-х или 3-х захватные)
- Гидравлические съемники
- Индукционные нагреватели

**Правила демонтажа:**
- Съем должен быть равномерным по всей окружности
- При горячей посадке — возможен нагрев корпуса для облегчения съема
- Не допускать повреждения вала и корпуса

## 3.7. Хранение, упаковка и переконсервация

### Условия хранения

**Требования к помещению:**
- Температура: +5...+25°C
- Влажность: не более 60%
- Отсутствие агрессивных паров
- Защита от прямых солнечных лучей

**Правила хранения:**
- Горизонтальное положение в оригинальной упаковке
- Защита от вибраций и ударов
- Исключение контакта с влагой
- Регулярный контроль состояния

### Консервация

**Типы консервации:**
- **Временная** (до 1 года) — масляная пленка
- **Средняя** (1-3 года) — пластичная смазка
- **Длительная** (3-5 лет) — специальные консервационные составы

**Переконсервация:**
- Необходима при истечении срока консервации
- Включает очистку от старой смазки и нанесение новой
- Проверка на отсутствие коррозии

**Упаковка:**
- **Индивидуальная** — для малых и средних подшипников
- **Групповая** — для крупных партий
- **Специальная** — для крупногабаритных подшипников

**Маркировка упаковки:**
- Обозначение подшипника
- Дата изготовления
- Срок консервации
- Особые условия хранения
