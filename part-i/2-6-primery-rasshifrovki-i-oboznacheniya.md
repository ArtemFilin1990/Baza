# 2.6. Примеры расшифровки и обозначения

*Дата последнего обновления: 2026-01-10*

---

### Простые обозначения

**Пример 1: 6205**
- **6** — серия диаметров: 2 (легкая)
- **2** — тип: 0 (шариковый радиальный однорядный)
- **05** — внутренний диаметр: 5 × 5 = 25 мм
- **Полная расшифровка:** Шариковый радиальный однорядный подшипник легкой серии с внутренним диаметром 25 мм

**Пример 2: 32310**
- **3** — серия ширин: нормальная
- **2** — тип: роликовый с короткими цилиндрическими роликами
- **3** — серия диаметров: средняя
- **10** — внутренний диаметр: 10 × 5 = 50 мм
- **Полная расшифровка:** Роликовый радиальный с цилиндрическими роликами, средней серии, d=50 мм

**Пример 3: 7208**
- **7** — тип: роликовый конический
- **2** — серия диаметров: легкая
- **08** — внутренний диаметр: 8 × 5 = 40 мм
- **Полная расшифровка:** Роликовый конический подшипник легкой серии с d=40 мм

**Пример 4: 180205**
- **1** — серия ширин: особо узкая
- **8** — тип: упорный шариковый
- **02** — серия диаметров
- **05** — внутренний диаметр: 25 мм
- **Полная расшифровка:** Упорный шариковый подшипник особо узкой серии, d=25 мм

### Обозначения с классом точности и зазором

**Пример 5: 6-180205**
- **6** — класс точности
- **180205** — основное обозначение
- **Полная расшифровка:** Упорный шариковый подшипник класса точности 6, d=25 мм

**Пример 6: 76-6205**
- **7** — группа радиального зазора (С3)
- **6** — класс точности
- **6205** — основное обозначение
- **Полная расшифровка:** Шариковый радиальный подшипник класса 6, с увеличенным зазором (группа 7), d=25 мм

**Пример 7: А5-46210**
- **А** — категория А (повышенные требования)
- **5** — класс точности 5
- **46210** — основное обозначение
- **Полная расшифровка:** Радиально-упорный шариковый подшипник категории А, класса точности 5, d=50 мм

**Пример 8: 125-205**
- **1** — ряд момента трения (минимальный)
- **2** — группа зазора 2 (С2)
- **5** — класс точности 5
- **205** — основное обозначение
- **Полная расшифровка:** Шариковый радиальный подшипник класса 5, группа зазора 2, с моментом трения по первому ряду, d=25 мм

### Обозначения с суффиксами

**Пример 9: 6205-2RS**
- **6205** — основное обозначение
- **2RS** — с двухсторонними резиновыми уплотнениями
- **Полная расшифровка:** Герметичный шариковый радиальный подшипник с двумя контактными уплотнениями, d=25 мм

**Пример 10: 6205NR**
- **6205** — основное обозначение
- **NR** — с канавкой и стопорным кольцом на наружном кольце
- **Полная расшифровка:** Шариковый радиальный подшипник с канавкой и стопорным кольцом для осевой фиксации, d=25 мм

**Пример 11: 32310А**
- **32310** — основное обозначение
- **А** — повышенной грузоподъемности
- **Полная расшифровка:** Роликовый цилиндрический подшипник повышенной грузоподъемности с увеличенным числом роликов, d=50 мм

**Пример 12: 6205Л**
- **6205** — основное обозначение
- **Л** — латунный сепаратор
- **Полная расшифровка:** Шариковый радиальный подшипник с массивным латунным сепаратором, d=25 мм

**Пример 13: 32310М**
- **32310** — основное обозначение
- **М** — модифицированный контакт
- **Полная расшифровка:** Роликовый цилиндрический подшипник с модифицированным контактом (бомбированные ролики), d=50 мм

**Пример 14: 6205Ю**
- **6205** — основное обозначение
- **Ю** — из нержавеющей стали
- **Полная расшифровка:** Шариковый радиальный подшипник из коррозионностойкой нержавеющей стали, d=25 мм

**Пример 15: 6308Т3**
- **6308** — основное обозначение
- **Т3** — температура отпуска 300°C
- **Полная расшифровка:** Шариковый радиальный подшипник для работы при температуре до +250°C, d=40 мм

### Сложные обозначения с несколькими суффиксами

**Пример 16: 76-6205Л С3**
- **7** — группа зазора 7 (С3)
- **6** — класс точности 6
- **6205** — основное обозначение
- **Л** — латунный сепаратор
- **С3** — увеличенный зазор (дублирование для ясности)
- **Полная расшифровка:** Шариковый радиальный подшипник класса точности 6, с латунным сепаратором, с увеличенным радиальным зазором группы С3, d=25 мм

**Пример 17: А5-46210К6М1У2**
- **А** — категория А
- **5** — класс точности 5
- **46210** — основное обозначение
- **К6** — конструктивное изменение (замок, угол 15°, текстолитовый сепаратор)
- **М1** — модифицированный контакт (вариант 1)
- **У2** — специальные технические требования (вариант 2)
- **Полная расшифровка:** Радиально-упорный шариковый шпиндельный подшипник категории А, класса точности 5, с замком на внутреннем кольце, углом контакта 15°, текстолитовым сепаратором, модифицированным контактом и специальными техтребованиями, d=50 мм

**Пример 18: 6205-2RSС17Ш2**
- **6205** — основное обозначение
- **2RS** — двухсторонние уплотнения
- **С17** — смазка ЦИАТИМ-221
- **Ш2** — очень низкий уровень вибрации
- **Полная расшифровка:** Герметичный малошумный шариковый радиальный подшипник с двумя уплотнениями, заполненный смазкой ЦИАТИМ-221, с очень низким уровнем вибрации, d=25 мм

**Пример 19: 32310АМТ4**
- **32310** — основное обозначение
- **А** — повышенной грузоподъемности
- **М** — модифицированный контакт
- **Т4** — температура отпуска 350°C
- **Полная расшифровка:** Роликовый цилиндрический подшипник повышенной грузоподъемности, с модифицированным контактом, для работы при температуре до +300°C, d=50 мм

**Пример 20: НТ18Ю**
- **НТ** — тугая подборка с пониженной точностью размеров
- **18** — серия
- **Ю** — из нержавеющей стали
- **Полная расшифровка:** Подшипник тугой подборки из нержавеющей стали с пониженной точностью размеров посадочных мест

### Специальные обозначения

**Пример 21: К25×30×10Д**
- **К** — игольчатый без колец
- **25×30×10** — d×D×B (мм)
- **Д** — сепаратор из алюминиевого сплава
- **Полная расшифровка:** Игольчатый подшипник без колец с алюминиевым сепаратором, d=25мм, D=30мм, B=10мм

**Пример 22: ИК15×27×16**
- **ИК** — игольчатый с внутренним кольцом
- **15×27×16** — d×D×B (мм)
- **Полная расшифровка:** Игольчатый подшипник с внутренним кольцом, d=15мм, D=27мм, B=16мм

**Пример 23: ШС40К**
- **Ш** — шарнирный подшипник
- **С** — для подвижных соединений с отверстиями для смазки
- **40** — внутренний диаметр 40 мм
- **К** — с отверстиями для смазки во внутреннем и наружном кольцах
- **Полная расшифровка:** Шарнирный подшипник для подвижных соединений с отверстиями для смазки в обоих кольцах, d=40мм
