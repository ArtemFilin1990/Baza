# 2.5. Конструктивные особенности и модификации (префиксы и суффиксы)

*Дата последнего обновления: 2026-01-10*

---

### Структура дополнительных знаков

**Дополнительные знаки** располагаются слева и справа от основного условного обозначения подшипника:

```
[Префиксы слева] - [Основное обозначение] [Суффиксы справа]
```

**Схема расположения знаков:**

| Позиция слева | Обозначение | Позиция справа | Обозначение |
|---|---|---|---|
| Категория | А, В, С | 1 | Повышенная грузоподъемность (А) |
| Момент трения | 1, 2, 3, 4 | 2 | Конструкция (Н, К...) |
| Зазор | С1-С5, Н | 3 | Материалы (И, Р, Х, Э, Ю, Я, Б, Г, Д, Е, Л) |
| Точность | 0, 6, 5, 4, 2, Т | 4 | Конструктивные изменения (К, К1...) |
| | | 5 | Модифицированный контакт (М, М1...) |
| | | 6 | Спецтребования (У, У1...) |
| | | 7 | Температура отпуска (Т, Т1-Т6) |
| | | 8 | Смазка (С1-С30) |
| | | 9 | Вибрация (Ш, Ш1-Ш5) |

**Пример полного обозначения:**  
`А76-46210К5М1У2Т3С17Ш2`
- А = категория А
- 7 = группа зазора 7
- 6 = класс точности 6
- 46210 = основное обозначение
- К5 = конструктивное изменение
- М1 = модифицированный контакт
- У2 = специальные требования
- Т3 = температура отпуска 300°C
- С17 = тип смазки
- Ш2 = уровень вибрации

### Префиксы (знаки слева от основного обозначения)

#### Категория подшипника

**Категории по ГОСТ 520:**

| Категория | Класс точности | Особенности |
|---|---|---|
| **А** | 5, 4, 2, Т | Повышенные требования к точности и качеству |
| **В** | 0, 6Х, 6, 5 | Дополнительные технические требования |
| **С** | 8, 7, 0, 6 | Стандартные требования (не обозначается) |

**Примеры:**
- `А5-205` — подшипник категории А, класса точности 5
- `В0-6205` — подшипник категории В, класса точности 0

#### Момент трения

**Обозначение:** Цифры 1, 2, 3, 4 перед обозначением зазора

| Ряд | Момент трения | Применение |
|---|---|---|
| **1** | Минимальный | Прецизионные шпиндели |
| **2** | Пониженный | Высокоскоростное оборудование |
| **3** | Средний | Стандартное применение |
| **4** | Повышенный | Тяжелонагруженные узлы |

**Особые обозначения:**
- **НТ** — тугая подборка с пониженной точностью размеров
- **НУ** — тугая подборка с пониженной точностью вращения

**Примеры:**
- `125-205` — момент трения ряд 1, зазор группа 2, класс 5
- `4М6-6205` — момент трения ряд 4, нормальный зазор, класс 6

#### Радиальный зазор

**Группы зазора (ГОСТ 24810):**

| Группа | ГОСТ | ISO | Величина зазора | Применение |
|---|---|---|---|---|
| **С1** | 1 | C1 | Меньше С2 | Прецизионные узлы |
| **С2** | 6, 2 | C2 | Малый | Стандартные применения |
| **CN** | (норма) | Normal | Нормальный | Большинство применений |
| **С3** | 7, 3 | C3 | Увеличенный | Высокие температуры |
| **С4** | 8, 4 | C4 | Большой | Очень высокие температуры |
| **С5** | 9, 5 | C5 | Очень большой | Экстремальные условия |
| **Н** | Н | - | Специальный | По техусловиям |

**Примечание:** Нормальная группа зазора в обозначении не указывается.

**Примеры:**
- `76-180607` — зазор группа 7, класс точности 6
- `Н0-42624М` — специальный зазор Н, класс 0

#### Класс точности

**Классы точности (в порядке повышения точности):**

| ГОСТ | ISO | ABEC | Допуск (мкм) | Применение |
|---|---|---|---|---|
| **8** | - | - | >15 | Неответственные узлы |
| **7** | - | - | 15-20 | Неответственные узлы |
| **0** | Normal (P0) | ABEC-1 | 13-15 | Общее машиностроение |
| **6Х** | - | - | 10-12 | Конические подшипники |
| **6** | P6 | ABEC-3 | 9-11 | Электродвигатели |
| **5** | P5 | ABEC-5 | 6-8 | Станки, насосы |
| **4** | P4 | ABEC-7 | 4-6 | Прецизионные станки |
| **2** | P2 | ABEC-9 | 2-3 | Высокоточное оборудование |
| **Т** | - | - | <2 | Сверхточные измерительные приборы |

**Примеры:**
- `6-6205` — класс точности 6
- `5-46210` — класс точности 5
- `Х-7208` — класс точности 6Х (для конических)

**Дополнительное обозначение:**
- **У** после класса точности — повышенная точность по монтажной высоте (конические подшипники)
- Пример: `6У-7608`

### Суффиксы (знаки справа от основного обозначения)

#### 1. Повышенная грузоподъемность (А)

**А** — повышенная грузоподъемность

**Для шариковых подшипников:**
- После 1996 г. все радиальные шариковые подшипники выпускаются с повышенной грузоподъемностью
- Обозначение А не имеет существенного значения

**Для роликовых подшипников:**
- Повышение достигается увеличением числа или диаметра роликов
- За счет утоньшения перемычек сепаратора
- Обозначение А существенно

**Примеры:**
- `6205А` — повышенная грузоподъемность
- `32310А` — роликовый с увеличенным числом роликов

#### 2. Конструктивные особенности (Н)

**Н** обозначает различные конструктивные особенности:

| Тип подшипника | Значение Н |
|---|---|
| Роликовые сферические двухрядные | Кольцевая проточка и 3 отверстия для смазки на наружном кольце |
| Роликовые цилиндрические без колец | Размеры по рекомендации СЭВ |
| Шариковые со сферической поверхностью | Углубление и отверстие для смазки |
| Упорные | Диаметр свободного кольца по ISO 104:1994 |

**Примеры:**
- `3620Н` — с проточкой и отверстиями
- `8316Н` — упорный с измененным диаметром свободного кольца

#### 3. Материалы деталей

**Материалы колец и тел качения:**

| Обозначение | Материал | Особенности | Применение |
|---|---|---|---|
| **И, И1, И2...** | Измененные ТУ | Изменение поставки комплектующих | Специальные требования |
| **Н, Н1, Н2...** | Теплостойкая сталь | Модифицированная жаропрочная сталь | Высокие температуры до +250°C |
| **Р, Р1, Р2...** | Быстрорежущая сталь | Теплоустойчивая сталь | До +300°C |
| **Х, Х1, Х2...** | Цементируемая сталь | Кольца и/или тела качения | Ударные нагрузки |
| **Э, Э1, Э2...** | ШХ15 со спецприсадками | Ванадий, кобальт, молибден | Повышенная износостойкость |
| **Ю, Ю1, Ю2...** | Нержавеющая сталь | Все детали или часть | Коррозионная стойкость |
| **Я, Я1, Я2...** | Редкие материалы | Керамика, стекло, твердые сплавы | Экстремальные условия |
| **W, W1, W2...** | Вакуумированная сталь | Особо чистая сталь | Высокая надежность |

**Материалы сепараторов:**

| Обозначение | Материал | Характеристики |
|---|---|---|
| **Б, Б1...** | Безоловянистая бронза | Высокая износостойкость |
| **Г, Г1...** | Черные металлы | Штампованный стальной |
| **Д, Д1...** | Алюминиевый сплав | Легкий, для высоких скоростей |
| **Е** | Пластмасса общая | Полимеры |
| **Е1** | Текстолит армированный | Высокие скорости |
| **Е5** | Стеклонаполненный полиамид | Малошумный, без смазки |
| **Л, Л1...** | Латунь | Массивный, высокие скорости |

**Соответствие ГОСТ ↔ ISO:**

| ГОСТ | ISO | Материал |
|---|---|---|
| Г | F | Черные металлы |
| Д | L | Алюминий |
| Л | M | Латунь |
| Е5 | TV, TN | Полиамид стеклонаполненный |
| Е, Е1 | T, TP | Текстолит |

**Примеры:**
- `6205Ю` — из нержавеющей стали
- `6305Л` — с латунным сепаратором
- `6205Е5` — с полиамидным сепаратором

#### 4. Конструктивные изменения (К)

**К, К1, К2...** — обозначение конструктивных изменений:

| Обозначение | Тип подшипника | Значение |
|---|---|---|
| **К, К1, К2** | Роликовые цилиндрические | Штампованный сепаратор из черных металлов |
| **К** | Радиально-упорные высоких классов | Замок на внутреннем кольце + текстолитовый сепаратор |
| **К** | Роликовые цилиндрические двухрядные | Проточка на наружном кольце + 3 отверстия для смазки |
| **К5** | Радиальные шариковые | Повышенная грузоподъемность |
| **К5** | Сферические роликовые | Сепаратор, базируемый по наружному кольцу |
| **К6** | Радиально-упорные высоких классов | Замок на внутреннем кольце, угол 15° + текстолитовый сепаратор |
| **К7** | Радиально-упорные высоких классов | Замок на внутреннем кольце, угол 12° + текстолитовый сепаратор |
| **К7** | Шариковые со сферической поверхностью | Двусторонние уплотнения + стопорный штифт |
| **К10** | Шариковые со сферической поверхностью | Улучшенная конструкция уплотнений + стопорный штифт |
| **К11** | Радиально-упорные | Штампованный сепаратор |

**Примеры:**
- `N206К` — с штампованным сепаратором
- `46210К6` — угол контакта 15°, текстолитовый сепаратор

#### 5. Модифицированный контакт (М)

**М, М1, М2...** — модифицированный контакт для роликовых подшипников

**Суть модификации:**
- Ролики с выпуклой центральной частью ("бомбина")
- Компенсация краевого эффекта при перекосах
- Оптимизация распределения нагрузок
- Увеличение срока службы

**Типы модификации:**
- **М** — стандартная модификация
- **М1** — первый вариант исполнения
- **М2** — второй вариант исполнения

**Примеры:**
- `32310М` — конический с модифицированным контактом
- `N308М1` — цилиндрический с модификацией М1

#### 6. Специальные технические требования (У)

**У, У1, У2, У3...** — специальные технические требования:

**Типы требований:**
- Ужесточенные требования по шероховатости
- Повышенная точность вращения
- Специальные требования к чистоте обработки
- Особые требования к радиальному зазору
- Специальные покрытия
- Дополнительный контроль качества

**Примеры:**
- `6205У` — со специальными требованиями
- `46210У2` — второй вариант спецтребований

#### 7. Температура отпуска (Т)

**Т, Т1-Т6** — температура стабилизирующего отпуска:

| Обозначение | Температура отпуска, °C | Рабочая температура, °C | Применение |
|---|---|---|---|
| **без обозначения** | 150 | до 120 | Стандартные условия |
| **Т** | 200 | до 160 | Повышенная температура |
| **Т1** | 225 | до 180 | Высокая температура |
| **Т2** | 250 | до 200 | Очень высокая температура |
| **Т3** | 300 | до 250 | Печи, сушилки |
| **Т4** | 350 | до 300 | Термооборудование |
| **Т5** | 400 | до 350 | Высокотемпературное оборудование |
| **Т6** | 450 | до 390 | Экстремальные температуры |

**Примеры:**
- `6205Т2` — рабочая температура до 200°C
- `32310Т4` — рабочая температура до 300°C

#### 8. Смазка (С)

**С1, С2...С30** — виды пластичных смазок для подшипников закрытого типа:

| Обозначение | Тип смазки | Температурный диапазон, °C | Применение |
|---|---|---|---|
| **С1** | Солидол жировой | -20...+60 | Низкоскоростные узлы |
| **С2** | Солидол синтетический | -40...+80 | Общего назначения |
| **С3** | Литол-24 | -40...+120 | Широкое применение |
| **С17** | Смазка ЦИАТИМ-221 | -60...+150 | Широкий диапазон температур |
| **С23** | Смазка высокотемпературная | -30...+180 | Высокотемпературные узлы |
| **С30** | Специальная смазка | По ТУ | Особые условия |

**Примеры:**
- `6205-2RSС3` — с уплотнениями, заполненный смазкой Литол-24
- `6308-2RSС17` — с уплотнениями, смазка ЦИАТИМ-221

#### 9. Уровень вибрации (Ш)

**Ш, Ш1-Ш5** — уровень вибрации (шумность):

| Обозначение | Уровень вибрации | Применение |
|---|---|---|
| **без обозначения** | Стандартный | Общее применение |
| **Ш** | Пониженный | Тихие механизмы |
| **Ш1** | Низкий | Малошумные применения |
| **Ш2** | Очень низкий | Прецизионное оборудование |
| **Ш3** | Минимальный | Измерительные приборы |
| **Ш4** | Сверхнизкий | Особо тихие применения |
| **Ш5** | Ультранизкий | Медицинское оборудование |

**Примечание:** С возрастанием цифрового индекса уровень вибрации уменьшается.

**Примеры:**
- `6205Ш` — пониженная вибрация
- `6308Ш3` — минимальная вибрация

### Специальные обозначения

#### Уплотнения и защитные шайбы

| Обозначение | Тип | Контакт | Герметичность |
|---|---|---|---|
| **Z** | Защитная шайба с одной стороны | Бесконтактная | Низкая |
| **ZZ (2Z)** | Защитные шайбы с двух сторон | Бесконтактная | Средняя |
| **RS** | Резиновое уплотнение с одной стороны | Контактное | Высокая |
| **2RS** | Резиновые уплотнения с двух сторон | Контактное | Очень высокая |
| **Н** | Уплотнение по ГОСТ | Контактное | Высокая |
| **НР** | Двухстороннее уплотнение + расширенное кольцо | Контактное | Очень высокая |

**Примеры:**
- `6205Z` — с одной защитной шайбой
- `6205-2RS` — герметичный с двумя уплотнениями

#### Стопорные элементы

| Обозначение | Тип | Применение |
|---|---|---|
| **N** | С канавкой на наружном кольце | Для стопорного кольца |
| **NR** | С канавкой + стопорное кольцо | Осевая фиксация |

**Примеры:**
- `6205N` — с канавкой
- `6205NR` — с канавкой и кольцом
