# 12.4. Шкивы (для ремней, включая конусные TAPER BUSH)

*Дата последнего обновления: 2026-01-10*

---

### Шкивы для клиновых ремней

**Параметры:**
- Профиль канавки (О, А, Б, В, Г, Д)
- Количество ручьев (1-8)
- Диаметр (измеряется по дну канавки)

**Материалы:**
- Чугун СЧ20 (стандартные)
- Сталь (высокие скорости)
- Алюминий (легкие, высокие скорости)

### Шкивы для поликлиновых ремней

**Параметры:**
- Профиль (PJ, PL, PM)
- Количество ручьев
- Диаметр

**Минимальные диаметры:**
- PJ: 20-40 мм
- PL: 50-100 мм

### Шкивы для зубчатых ремней

**Параметры:**
- Шаг зубьев (T5, T10, AT10, HTD и т.д.)
- Количество зубьев
- Ширина

**Материалы:**
- Алюминий (легкие)
- Сталь (тяжелые нагрузки)

### Крепление TAPER BUSH

**Преимущества:**
- Быстрая установка/снятие
- Не нужна шпонка
- Точное центрирование
- Возможность установки на гладкий вал

**Процесс монтажа:**
1. Очистить контактные поверхности
2. Вставить втулку в ступицу шкива
3. Совместить отверстия
4. Затянуть болты крест-накрест
5. Проверить биение

**Типоразмеры TAPER BUSH:**
- 1008, 1108, 1210, 1215
- 2012, 2517, 3020, 3030
- 3535, 4040, 4545, 5050
