# 13.1. Стопорные кольца

*Дата последнего обновления: 2026-01-10*

---

### Типы стопорных колец

**1. Наружные (для валов):**
- **DIN 471** — стандартные для валов
- **Обозначение:** A × d
- Пример: A10 — для вала Ø10 мм

**2. Внутренние (для отверстий):**
- **DIN 472** — стандартные для отверстий
- **Обозначение:** J × D
- Пример: J20 — для отверстия Ø20 мм

**3. Упорные (осевые):**
- **DIN 6799** — тонкие упорные шайбы
- Для небольших осевых нагрузок

**Материалы:**
- **Пружинная сталь** — стандартные условия
- **Нержавеющая сталь** — коррозионная стойкость
- **Бериллиевая бронза** — немагнитные применения

**Установка:**
- Специальные щипцы (раздвижные для наружных, сжимающие для внутренних)
- Канавка должна быть чистой
- Проверка полной посадки в канавку

**Применение:**
- Фиксация подшипников
- Ограничение хода деталей
- Фиксация шестерен на валах
- Удержание сборочных узлов
