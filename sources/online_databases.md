# Онлайн базы данных и каталоги подшипников

## Описание

Этот документ содержит список информационных сайтов для базы знаний по подшипникам, включающий таблицы размеров, каталоги, поиск по параметрам и официальные ресурсы производителей.

---

## Каталоги с техническими данными / размерными таблицами

### 1. podshipnik.info
**Описание:** Каталог подшипников с размерами, массой и маркировками для ГОСТ-подшипников.

**URL:** [podshipnik.info - Каталог подшипников ГОСТ](https://podshipnik.info/catalog/katalog-podshipnikov-gost)

**Особенности:**
- Полный каталог подшипников по ГОСТ
- Размерные таблицы (d, D, B)
- Масса подшипников
- Маркировки по ГОСТ

**Применение:** Основной справочник для подбора подшипников по российским стандартам.

---

## Рекомендации по использованию

### Приоритет источников

1. **Официальные каталоги производителей** (SKF, NTN, NSK, Timken)
   - Первоисточники технических спецификаций
   - Актуальные размеры и характеристики
   - Официальная документация
   - Используйте для критических применений

2. **Справочные порталы** (podshipnik.info, IRBIS, kugellager-express.de)
   - Агрегированные таблицы размеров
   - Таблицы соответствий ISO/GOST
   - Быстрый поиск по параметрам
   - Используйте для проверки и расширения базы

3. **Инструменты подбора** (2rs.com.ua, podshipnikon.ru)
   - Интерактивный поиск
   - Подбор по геометрическим размерам
   - Быстрое определение типоразмера

### Методика работы

1. **Начните с официальных данных** как первоисточников
2. **Сверьте с независимыми каталогами** для проверки
3. **Используйте таблицы соответствий** для поиска аналогов
4. **Проверьте размеры** по нескольким источникам
5. **Документируйте источник** каждой спецификации

### Проверка достоверности

При использовании данных из онлайн-источников:
- Сравните информацию минимум из двух независимых источников
- При критических применениях используйте только официальные каталоги производителей
- Учитывайте дату публикации каталога
- Проверяйте соответствие стандартам (ГОСТ/ISO)
