*Дата последнего обновления: 2026-01-10*

## 9.1. Российские/СНГ: ГПЗ, МПЗ и другие заводы

**Основные производители:**

### ГПЗ (Государственные подшипниковые заводы)

**Крупнейшие заводы:**
- **ГПЗ-1 (Москва)** — универсальные подшипники
- **ГПЗ-3 (Самара)** — автомобильные подшипники
- **ГПЗ-4 (Куйбышев)** — роликовые подшипники
- **ГПЗ-5 (Курск)** — малогабаритные подшипники
- **ГПЗ-9 (Самара)** — игольчатые подшипники
- **ГПЗ-10 (Ростов-на-Дону)** — крупногабаритные подшипники
- **ГПЗ-23 (Вологда)** — роликовые конические

### МПЗ (Московский подшипниковый завод)
- Универсальные подшипники
- Прецизионные подшипники
- Экспортная продукция

### Другие заводы СНГ:
- **ВПЗ (Волжский)** — шариковые и роликовые
- **СПЗ (Саратовский)** — специальные подшипники
- **Минский подшипниковый завод** — универсальные

**Маркировка:**
- Обозначение по ГОСТ
- Код завода-изготовителя
- Год выпуска (буквенный код)

## 9.2. Международные: SKF, FAG/INA, NSK, NTN-SNR, TIMKEN, KOYO

### SKF (Швеция)

**История:** Основана в 1907 году, крупнейший производитель в мире

**Особенности маркировки:**
- Базовое обозначение (6205, NU 308)
- Префиксы (GE, YAR)
- Суффиксы (-2Z, -2RS1, C3)

**Специальные серии:**
- **Explorer** — повышенная производительность
- **E2** — энергоэффективные
- **CARB** — тороидальные роликовые

### FAG/INA (Германия, Schaeffler Group)

**История:** FAG — 1883, INA — 1946, объединены в Schaeffler

**Особенности:**
- **FAG** — универсальные подшипники
- **INA** — игольчатые и линейные направляющие

**Маркировка:**
- Стандартное ISO обозначение
- Дополнительные суффиксы (-2RSR, -TVH)

**Специальные серии:**
- **X-life** — увеличенный ресурс
- **ARCANOL** — специальные смазки

### NSK (Япония)

**История:** Основана в 1916 году

**Особенности маркировки:**
- HR — высокая грузоподъемность
- V — вибростойкие
- DDU — двойное уплотнение

**Специальные серии:**
- **Molded-Oil** — твердая смазка
- **High-Speed** — высокие скорости

### NTN-SNR (Япония/Франция)

**История:** NTN — 1918, SNR — 1916, объединение в 2008

**Особенности:**
- Автомобильные подшипники
- Прецизионные подшипники

### TIMKEN (США)

**История:** Основана в 1899 году

**Специализация:**
- **Конические роликовые подшипники**
- Крупногабаритные подшипники
- Промышленные применения

**Маркировка:**
- Собственная система обозначений
- Дюймовые размеры

### KOYO (Япония)

**История:** Основана в 1921 году

**Особенности:**
- Автомобильные подшипники
- Игольчатые подшипники
- Комплектующие для OEM

## 9.3. Особенности маркировки каждого бренда

### SKF

**Структура обозначения:**
```
[Префикс] - [Базовое обозначение] - [Суффикс]
```

**Примеры префиксов:**
- **GE** — сферические подшипники скольжения
- **YAR** — Y-подшипники с адаптером
- **CARB** — тороидальные роликовые

**Примеры суффиксов:**
- **-2Z** — две металлические шайбы
- **-2RS1** — два контактных уплотнения
- **C3** — радиальный зазор больше нормального
- **/C3** — зазор C3
- **-E2** — энергоэффективный

### FAG

**Суффиксы:**
- **-2RSR** — два резиновых уплотнения
- **-2Z** — две защитные шайбы
- **-M** — массивная латунная обойма
- **-TVH** — полиамидная обойма

**Префиксы:**
- **B** — угол контакта 40°
- **QJ** — четырехточечный контакт

### NSK

**Суффиксы:**
- **ZZ** — две металлические защиты
- **DDU** — два уплотнения (Double Dust seal)
- **CM** — радиальный зазор
- **V** — вибростойкий

**Префиксы:**
- **HR** — высокая грузоподъемность
- **RHP** — прецизионный

## 9.4. Как пользоваться онлайн-каталогами

### Основные онлайн-каталоги:

**SKF:**
- skf.com — официальный сайт
- Функция подбора по размерам
- 3D-модели CAD
- Технические данные

**FAG/Schaeffler:**
- schaeffler.com — глобальный каталог
- medias.schaeffler.com — техническая документация
- Калькуляторы ресурса

**NSK:**
- nsk.com — официальный каталог
- Поиск по обозначению
- Кросс-референс с другими брендами

**Bearing Distributors:**
- Многобрендовые каталоги
- Сравнение характеристик
- Наличие и цены

### Как искать:

**1. Поиск по обозначению:**
- Введите полное обозначение
- Используйте кросс-референс для аналогов

**2. Поиск по размерам:**
- Внутренний диаметр (d)
- Наружный диаметр (D)
- Ширина (B)

**3. Поиск по применению:**
- Тип оборудования
- Условия эксплуатации
- Требования к характеристикам

**4. Использование фильтров:**
- Тип подшипника
- Производитель
- Класс точности
- Наличие уплотнений

### Полезные функции каталогов:

- **CAD-модели** — скачивание 3D-моделей
- **Технические таблицы** — грузоподъемность, размеры
- **Инженерные калькуляторы** — расчет ресурса
- **Подбор аналогов** — замена недоступных подшипников
- **Руководства по монтажу** — инструкции
