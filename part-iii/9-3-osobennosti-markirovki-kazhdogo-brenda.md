# 9.3. Особенности маркировки каждого бренда

*Дата последнего обновления: 2026-01-10*

---

### SKF

**Структура обозначения:**
```
[Префикс] - [Базовое обозначение] - [Суффикс]
```

**Примеры префиксов:**
- **GE** — сферические подшипники скольжения
- **YAR** — Y-подшипники с адаптером
- **CARB** — тороидальные роликовые

**Примеры суффиксов:**
- **-2Z** — две металлические шайбы
- **-2RS1** — два контактных уплотнения
- **C3** — радиальный зазор больше нормального
- **/C3** — зазор C3
- **-E2** — энергоэффективный

### FAG

**Суффиксы:**
- **-2RSR** — два резиновых уплотнения
- **-2Z** — две защитные шайбы
- **-M** — массивная латунная обойма
- **-TVH** — полиамидная обойма

**Префиксы:**
- **B** — угол контакта 40°
- **QJ** — четырехточечный контакт

### NSK

**Суффиксы:**
- **ZZ** — две металлические защиты
- **DDU** — два уплотнения (Double Dust seal)
- **CM** — радиальный зазор
- **V** — вибростойкий

**Префиксы:**
- **HR** — высокая грузоподъемность
- **RHP** — прецизионный
