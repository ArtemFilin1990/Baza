# 1.1.5 Категории подшипников

## Классификация по категориям

### По направлению нагрузки

1. **Радиальные** — воспринимают радиальную нагрузку
   - Шариковые однорядные (6xxx)
   - Роликовые цилиндрические (Nxxx)
   
2. **Упорные** — воспринимают осевую нагрузку
   - Шариковые упорные (8xxx)
   - Роликовые упорные (9xxx)

3. **Радиально-упорные** — комбинированная нагрузка
   - Конические (7xxx)
   - Радиально-упорные шариковые (36xxx)

### По типу тел качения

- **Шариковые** — шарики
- **Роликовые** — цилиндрические, конические, бочкообразные ролики
- **Игольчатые** — иглы (тонкие ролики)

### По количеству рядов

- Однорядные
- Двухрядные  
- Многорядные
