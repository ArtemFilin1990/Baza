# 1.3.5 Классификация подшипников

## По типу тел качения

### Шариковые
- Низкое трение
- Высокие скорости
- Умеренные нагрузки

### Роликовые
- Высокие нагрузки
- Жесткость
- Средние скорости

### Игольчатые
- Компактные
- Высокие радиальные нагрузки
- Без осевых нагрузок

## По направлению нагрузки

| Тип | Радиальная | Осевая | Код |
|-----|-----------|--------|-----|
| Радиальные | +++  | + | 6xxx, 2xxx |
| Упорные | - | +++ | 8xxx, 9xxx |
| Радиально-упорные | ++ | ++ | 7xxx, 36xxx |

## По количеству рядов

- **Однорядные**: компактные, стандарт
- **Двухрядные**: повышенная нагрузка
- **Многорядные**: специальные применения

## По конструкции

- **Разъемные**: конические (раздельные кольца)
- **Неразъемные**: радиальные шариковые
- **Самоустанавливающиеся**: сферические
