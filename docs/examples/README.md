# Примеры статей, созданных по промту

Эта директория содержит примеры готовых статей, созданных с использованием промта [`PROMPT_ARTICLE_CREATION.md`](../../PROMPT_ARTICLE_CREATION.md).

## Цель примеров

Примеры демонстрируют:
- ✅ Правильную структуру статьи
- ✅ Оформление метаданных и навигации
- ✅ Использование таблиц и примеров
- ✅ Формулировку ключевых выводов
- ✅ Организацию связанных материалов

## Доступные примеры

### Теоретические статьи (Часть I)

#### [example-article-1-1.md](./example-article-1-1.md) - "Что такое подшипник? Основные функции"

**Тип:** Теоретическая статья  
**Объем:** ~3200 слов  
**Таблиц:** 7  
**Примеров:** 3  

**Что демонстрирует:**
- Полное определение с технической точностью
- Историческая справка
- Физические принципы работы с формулами и расчетами
- Сравнительные таблицы (подшипники качения vs скольжения)
- Практические примеры применения
- Примеры расчетов и экономического эффекта
- Расшифровку обозначения подшипника

**Структура:**
```markdown
# Заголовок + метаданные
## Содержание
## 8 основных разделов с подразделами
## Ключевые выводы (5 пунктов)
## Связанные материалы (ссылки)
```

---

## Как использовать примеры

### Шаг 1: Изучите структуру

Откройте пример и обратите внимание на:
- Оформление заголовка и метаданных
- Структуру содержания (оглавления)
- Логику изложения материала
- Форматирование таблиц
- Оформление примеров
- Формулировку ключевых выводов

### Шаг 2: Адаптируйте под свою тему

Используйте пример как шаблон:
- Сохраните общую структуру
- Замените контент на свою тему
- Добавьте специфичные для вашей темы разделы
- Убедитесь, что соблюдены все требования промта

### Шаг 3: Проверьте качество

Используйте [чек-лист из руководства](../ARTICLE_CREATION_GUIDE.md#проверочный-чек-лист):
- [ ] Все обязательные элементы присутствуют
- [ ] Объем статьи соответствует требованиям
- [ ] Минимум 2 таблицы
- [ ] Есть практические примеры
- [ ] Указаны источники
- [ ] Работают все ссылки

## Сравнение типов статей

| Элемент | Теоретическая (Часть I) | Справочная (Части II-IV) |
|---------|-------------------------|--------------------------|
| **Фокус** | Понимание принципов | Практическое применение |
| **Объем** | 2000-3000 слов | 1500-2500 слов |
| **Определения** | Полные с источниками | Краткие технические |
| **История** | Обязательна | Опциональна |
| **Формулы** | При необходимости | Редко |
| **Таблицы** | Сравнительные | Классификационные, справочные |
| **Примеры** | Концептуальные | Практические (расшифровка, подбор) |
| **Ключевые выводы** | 4-5 пунктов | 3-4 пункта |
