# Курс для менеджеров по подбору подшипников

## Цель курса
Дать менеджерам по продажам и снабжению базовые знания о подшипниках, чтобы уверенно консультировать клиентов и подбирать аналоги.

## 1. Основы
- **Типы подшипников**: шариковые (радиальные, радиально‑упорные, упорные), роликовые (с короткими цилиндрическими, коническими, сферическими роликами), игольчатые.
- **Классификация**: по направлению нагрузки (радиальные / упорные / радиально‑упорные), по числу рядов (одинарные / двойные), по возможности самоустановки (сферические).
- **Маркировка ГОСТ и ISO**: основные обозначения (серия, тип, диаметр), префиксы и суффиксы. Помните, что последние две цифры номера ×5 дают внутренний диаметр【371647085035825†L67-L148】.

## 2. Алгоритм подбора
1. Определите условия работы узла: тип нагрузки, обороты, температура, ресурс.
2. Подберите типоразмер по ГОСТ, используя каталоги или документы заказчика.
3. Найдите аналог по ISO через таблицу соответствия ( см. раздел `analogues`) и проверьте размеры, зазор и класс точности.
4. Подберите производителя исходя из требований к качеству и бюджета:
   - **Премиум**: для ответственных узлов и высокой нагрузки【271470310052723†L0-L18】.
   - **Средний сегмент**: хорошее соотношение цена/качество.
   - **Бюджет**: для типовых узлов без высокой нагрузки【271470310052723†L112-L132】.
5. Уточните наличие уплотнений, смазки и дополнительных требований (тепловой зазор, материал сепаратора).

## 3. Частые ошибки и как их избежать
- **Фокус только на цене**: дешёвый подшипник с низким ресурсом может привести к остановке оборудования.
- **Заказ без учёта бренда**: «неизвестные» бренды часто не отвечают стандартам, что повышает риск рекламаций.
- **Отсутствие проверки зазора и класса точности**: неправильный зазор приводит к перегреву или шуму【371647085035825†L67-L148】.
- **Покупка бывших в употреблении подшипников**: ресурс и качество неизвестны.
